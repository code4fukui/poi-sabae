## 機能
- 福井県鯖江市の観光スポットを対話型のマップとビューアーで表示します
- 曜日と時間帯によるフィルター機能で営業中の場所を表示できます
- 市の公開データを利用しています

## 使用方法
1. [デモページ](https://code4fukui.github.io/poi-sabae/)を開きます。
2. チェックボックスとドロップダウンメニューを使って、曜日と時間帯でフィルターしたPOIを表示します。
3. 地図上のマーカーやテーブルの行をクリックすると、各施設の詳細情報が表示されます。

## データ / API
POIのデータは、`poi-sabae.csv`ファイルに保存されており、CSV.jsライブラリを使って取得されます。
